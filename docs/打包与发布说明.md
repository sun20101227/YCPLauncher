# YCP Launcher 打包与发布

当前正式版本：**1.1.8**

版本号统一维护在根目录 `Directory.Build.props`。客户端、安装器、插件和发布脚本会自动读取该版本，不再需要逐个文件修改。

## 构建安装程序

```powershell
.\scripts\build.ps1
```

或双击 `scripts\1-构建安装程序.bat`。

输出：

- `artifacts\installer\YCPInstaller.exe`
- `artifacts\portable\YCPLauncher\YCPLauncher.exe`：免安装主程序
- `artifacts\YCPLauncher_Portable_v1.1.8.zip`：便携版压缩包
- `artifacts\app\`：安装器内部负载

## 打包名字同步插件

```powershell
.\scripts\package-plugin.ps1
```

输出：`artifacts\YCPNameSyncPlugin_v1.1.8.zip`

## 发布 GitHub Release

1. 安装 GitHub CLI。
2. 执行 `gh auth login`。
3. 执行 `.\scripts\build.ps1`，生成安装程序和便携版。
4. 执行 `.\scripts\package-plugin.ps1`，生成名字同步插件包。
5. 确认 `docs\release-notes\1.1.8.md` 内容。
6. 执行 `.\scripts\release-github.ps1`。

脚本会自动读取版本号和对应版本的发布说明，并将以下文件上传到同一个 GitHub Release：

- `artifacts\installer\YCPInstaller.exe`
- `artifacts\YCPLauncher_Portable_v1.1.8.zip`
- `artifacts\YCPNameSyncPlugin_v1.1.8.zip`
