# YCP Launcher 1.1.8

本次更新聚焦于界面品质、直播稳定性、整体性能和 CS2 一键进服可靠性。

## 全新视觉体验

- 重构深色与浅色主题，采用更克制的石墨灰、深海军蓝与信号青配色。
- 重新设计登录页、比赛房间入口和房间大厅的信息层级。
- 将重复的大型席位按钮改为轻量次级操作，减少视觉干扰。
- 优化背景动态壁纸的透明度和主题遮罩，兼顾氛围与文字可读性。
- 统一页面标题、卡片、弹窗、Toast 和导航视觉。
- 修复 TextBox、PasswordBox、ComboBox 及房间码输入框文字未垂直居中的问题。

## 直播稳定性

- 修复直播页在深浅主题切换时闪烁的问题。
- 修复关闭或最小化直播页面后残留 `LibVLCSharp.WPF` 原生窗口的问题。
- 加强播放器初始化、刷新、超时和页面切换的生命周期管理。
- 进入直播页面时暂停背景视频，避免同时运行两套视频解码器。
- 最小化和隐藏到托盘时停止直播拉流与视频解码。

## 性能优化

- 合并全窗口视频遮罩并减少大面积透明合成。
- 移除昂贵的布局动画和大范围模糊阴影。
- 统一为短时、轻量的页面过渡动画，并完整支持“减少动画”设置。
- 优化 Reveal 光效、平滑滚动、Dashboard 轮询和大厅槽位刷新。
- 修复注销、页面切换和退出过程中可能残留的后台任务与资源。

## CS2 一键进服

- 新增 cfg 预写入、启动检测和连接补发的多阶段进服流程。
- 修复部分情况下只能启动 CS2、无法自动加入服务器的问题。
- 加强 Steam 与 CS2 多库安装路径识别。
- 修复 Steam URI 中 `+connect` 被错误编码的问题。
- 连续选择不同服务器时，旧请求不会再把玩家连接回之前的服务器。
- 加固服务器地址和启动参数校验，防止控制台命令注入。

## 工程整理

- 客户端、安装器、卸载器、插件和工具迁移到统一目录结构。
- 版本号集中维护于 `Directory.Build.props`。
- 统一安装包、便携版、插件包和 GitHub Release 构建流程。
- 移除旧构建产物、重复源码和已废弃的历史目录。
