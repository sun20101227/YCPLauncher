## 更新内容

### 新功能
- **一键进服模式自由切换**：在“设置 - 高级性能与协议”中新增了“一键进服协议”下拉选项。
  - **模式 1 (推荐)**：使用最底层、最稳定的原生协议（完美解决无响应、闪退问题）。
  - **模式 2**：传统的文件注入方式。
  - **模式 3**：通用连接协议备用。
  如果遇到进服问题，请任意切换模式尝试。

### 优化与修复
- **全并行竞速测速下载**：更新器彻底重构了网络节点检测逻辑。现在不再是傻乎乎地按顺序等待超时，而是**同时向所有代理节点和 GitHub 官方直连发起网络请求。哪个节点连通性最好、最先响应，更新器就会光速采纳该节点并自动掐断其他慢节点。** 彻底告别网络报错与卡顿！
